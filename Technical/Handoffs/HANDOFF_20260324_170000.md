## Critical Warnings

1. **FDIC API domain**: Use `banks.data.fdic.gov`, NOT `api.fdic.gov`
2. **FDIC API offset limit**: Max offset is 2,000,000. Use year filters for large tables.
3. **FFIEC CDR**: Blocked by Cloudflare. Do not attempt programmatic download.
4. **FFIEC NPW**: Also blocked by Cloudflare as of 2026-03-24. Try browser download.
5. **DuckDB temp files**: COPY TO creates tmp_ files that persist if process crashes.
6. **Luck Database variable names**: Use actual names (e.g., `assets`, `deposits`, `ytdnetinc`), NOT guessed names.
7. **BHCF prefix variants**: BHCK=consolidated, BHCP=parent only, BHCT=total, etc. Use BHCK for most analyses.
