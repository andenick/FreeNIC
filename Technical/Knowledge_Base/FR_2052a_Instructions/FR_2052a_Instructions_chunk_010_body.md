## Appendix II-c: FR 2052a Collateral Class Requirements

The following table displays the applicable reporting requirements for the Collateral Class field by Product:

1. **Required**: the product by definition requires a collateral class designation
2. **Dependent**: the product requires a collateral class designation to the extent collateral has been, or would need to be posted or received
3. **Not applicable**: the product should not be assigned a collateral class

## Appendix III: FR 2052a Asset Category Table

Note: the "-Q" suffix indicates that assets meet all the asset-specific tests detailed in section 20 of Regulation WW (e.g., risk profile and market-based characteristics).

The Asset Category Table defines all collateral class codes used across the FR 2052a report, organized into three tiers:

- **HQLA Level 1**: A-0-Q (Cash), A-1-Q through A-5-Q (U.S. Government), S-1-Q through S-4-Q (Non-U.S. Sovereign), CB-1-Q through CB-2-Q (Central Bank)
- **HQLA Level 2a**: G-1-Q through G-3-Q (GSE), S-5-Q through S-7-Q (Non-U.S. Sovereign 20% RW), CB-3-Q (Central Bank 20% RW)
- **HQLA Level 2b**: E-1-Q, E-2-Q (Equities), IG-1-Q, IG-2-Q (Investment Grade)
- **Non-HQLA**: Same codes without "-Q" suffix (A-2 through CB-3, G-1 through G-3, S-5 through S-7, CB-3, E-1 through E-2, IG-1 through IG-2)

See FR_2052a_Instructions_chunk_010_table_004.csv for the complete HQLA asset category table and FR_2052a_Instructions_chunk_010_table_005.csv for the Non-HQLA asset category table.
