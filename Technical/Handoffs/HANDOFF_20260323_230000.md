## Critical Warnings

1. **FDIC API domain**: Use `banks.data.fdic.gov`, NOT `api.fdic.gov`
2. **FDIC API offset limit**: Max offset is 2,000,000. Use year filters for large tables.
3. **FFIEC CDR**: Blocked by Cloudflare. Do not attempt programmatic download.
4. **DuckDB temp files**: COPY TO creates tmp_ files that persist if process crashes. Check for and clean up orphaned processes.
5. **Luck Database dates**: Quarter-start datetimes (Jan 1, Apr 1, Jul 1, Oct 1), NOT sequential integers. Fixed in current script.
